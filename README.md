# user-funnel-analysis
User behavior and funnel analysis to improve engagement metrics

Background
The objective of this analysis was to understand uneven engagement trends despite stable traffic growth. The goal was to identify drop-off drivers and inform product improvements.

Approach
Performed user-level segmentation
Conducted funnel analysis across key stages
Identified early disengagement patterns
Evaluated engagement metrics before and after UX changes

Tools
Python, SQL, pandas, matplotlib

Outcome
Insights informed product and UX improvements, resulting in:
60% increase in page views
35% increase in session duration
